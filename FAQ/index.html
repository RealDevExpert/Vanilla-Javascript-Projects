Frequently Asked Questions

Is JavaScript a case-sensitive language?

Yes, JavaScript is a case sensitive language.

What is the purpose of ‘This’ operator in JavaScript?

JavaScript this keyword refers to the object it belongs to.In a method, this refers to the owner object and in a function, this refers to the global object

What is Callback?

A callback is a plain js function passed to some method as an argument or option. It's a function that is to be executed after another function has finished executing.

How to create a cookie using JavaScript?

The simplest way to create a cookie is to assign a string value to the document.cookie object, which looks like this-
Syntax
document.cookie = "key1 = value1; key2 = value2; expires = date";
